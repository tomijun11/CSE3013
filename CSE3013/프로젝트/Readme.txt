-- # 프로젝트 실행방법 ------------------------------------
프로젝트의 src폴더에 source폴더에 있는 소스코드들을 모두 넣고 프로젝트에서 로드해주세요
프로젝트의 bin/data폴더에 data폴더에 있는 것(.maz, .png, .ttf)들을 모두 넣어주세요

-- # 실행환경 ---------------------------------------------
OS : Windows 10 Home
CPU : Intel(R) Core(TM) i7-6700HQ CPU @ 2.60GHz  2.60 GHz
RAM : 16.00GB
Compiler : Visual Studio 2022 Debug Mode

-- # 게임 플레이 방법 ----------------------------------
게임 방법 : 제한시간 내로 미로를 탈출하는 게임입니다.
캐릭터 (분홍색 사각형)을 방향키로 움직여 제한시간내로 도착점(초록색 원)에 도달하면 승리합니다.
제한시간 내로 도착점에 도달하지 못하면 패배합니다.

스테이지마다 점수에 따라 최대 3개의 코인을 얻을 수 있습니다.
스테이지는 총 5개이며, 이전 스테이지를 클리어하면 다음 스테이지에 도전할 수 있습니다. (해금된 스테이지는 흰색으로 표시됨)
스테이지 선택화면에서 각 스테이지의 정보에는 최고기록이 기록됩니다. (최단시간, 최대코인 갯수)
스테이지 클리어 및 최고기록 정보는 자동으로 bin/data폴더에 save.txt파일로 저장되고, 프로젝트 재실행시 로드됩니다.

점수 계산방법 : 남은 시간이 많을수록, 미로 내의 동전을 많이 먹을수록 점수가 올라가며, 힌트를 사용한 경우 점수의 절반이 감점됩니다.
동전(노란색 원)은 매 스테이지마다 랜덤 위치에 3개씩 생성됩니다.

장애물 : 파란색 물줄기에 닿으면 5초간 캐릭터가 둔화상태에 걸립니다. (오른쪽 Slowdown 막대로 남은 시간 확인가능)
빨간색 용암에 닿으면 즉시 게임오버 됩니다.
물줄기와 용암은 시간에 따라 일정한 주기를 가지고 켜졌다 껐다를 반복합니다.

-- # 조작법 ----------------------------------
조작법 (인게임) : 상하좌우 방향키로 캐릭터를 움직일 수 있습니다. (단, 둔화상태인 경우에는 절반의 속도로 움직임)
H키를 눌러 현재 위치부터 도착지까지의 경로를 10초간 출력할 수 있습니다. (게임당 1회 제한)
Q키를 눌러 스테이지 선택창으로 되돌아 갈 수 있습니다.

조작법 (메뉴) : 버튼 클릭 및 키보드 입력을 모두 지원하며, Q를 누르면 뒤로 돌아갈 수 있습니다.
스테이지 선택창에서 1~5번키를 눌러 각 스테이지를 플레이 할 수 있습니다. (단, 해금된 스테이지만 가능함)
